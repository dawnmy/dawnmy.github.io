Projects

1. QuasiModo - benchmarking of strain-resolved analysis for population of large DNA viruses

We developed a benchmarking tool QuasiModo and extensively evaluated 15 assemblers and five six variant callers on ten lab-generated benchmark data sets created with two different library preparation protocols, to identify best practices and challenges for analyzing such data. (Contribution: developed the software, benchmarked all methods in the study)

2. Deep neural networks optimized for genomic datasets

We developed a software based on deep learning (Tensorflow, Pytorch) to learn genomic patterns from sequence data. It can be adapted into different use cases, including sequence classification, gene prediction, genomic structure identification, et. al. (Contribution: implemented data visualization, built classification model)

3. Development of software for accurate pathogen detection and taxonomy profiling on long reads metagenomics data

We are developing computational method using Python, Snakemake and C++ for high resolution pathogen and taxonomy profiling. The software will predict antibiotic resistance for the identified pathogens for precision medicine.

4. Second round of challenges for Critical Assessment of Metagenome Interpretation (CAMI)

We established a set of benchmarking criteria, tool-kits and generated datasets for the assessment of metagenomic data anaylsis methods used for genome assembly, binning and taxonomy profiling. (Contribution: evaluated pathogen detection and profiling methods)

5. Explore biologically relevant chemical space for drug discovery (2011 ~ 2013)

We proposed a novel multi-dimensional molecular descriptor termed Biological Relevance Spectrum (BRS). Several prediction models (ADME/T and biological activity) built based on BRS using SVM reach both high Precision and Recall. (Contribution: data collection, built classification models)

6. AutoChrom3D: a webserver for modeling and visualization the 3D structure of human or mouse chromatin

We built a webserver to predict the 3D structure of chromatin based on Hi-C, 3C, 4C data. (Contribution: built the webserver)
