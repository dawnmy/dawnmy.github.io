Data visualization with Python

Oct 9, 2016 | Hits

最近经常用Python画柱状图和heatmap，一般需要存储为矢量图的PDF文件，方便后期编辑。这里面有些需要注意的地方，记录如下：

  • 关于pdf的图中启用Truetype字体方便编辑
  • barplot的legend大小位置设置
  • heatmap的colorbar的大小调整
  • 轴刻度标签对齐方式
